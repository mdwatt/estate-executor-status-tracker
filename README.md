# Estate Executor Status Tracker

> One centralized dashboard helping families and estate executors manage estate files with clarity and confidence.

## Overview

Estate Executor Status Tracker solves the scattered-information problem that plagues estate administration. Instead of hunting across emails, spreadsheets, and physical files, executors and authorized family members get a single, organized view of every task, deadline, and document in the estate process.

## Tech Stack

- Plain HTML / CSS / Vanilla JS — no frameworks, no build step
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
- [Font Awesome 6](https://fontawesome.com/) for icons
- [CountAPI](https://countapi.xyz/) for the live visitor counter
- GitHub Pages for hosting

## Deployment

Push any changes to the `main` branch. GitHub Pages publishes automatically from the root of the repo.

```bash
git add .
git commit -m "Update"
git push origin main
```
